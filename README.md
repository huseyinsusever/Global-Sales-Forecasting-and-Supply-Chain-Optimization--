# Global-Sales-Forecasting-and-Supply-Chain-Optimization--

### 🥤 Strategy Simulation for Beverage Industry Leaders

This project focuses on predicting sales volumes across 3,000+ retail locations to optimize inventory management and distribution logistics. 

**Key Objectives:**
- Analyze the impact of public holidays (like Ramadan and National Days) on sales spikes.
- Optimize supply chain efficiency by reducing out-of-stock scenarios.
- Implement predictive models to handle seasonality in the retail sector.

*Tools: Python, Pandas, Matplotlib, Seaborn, Rossmann Data.*
